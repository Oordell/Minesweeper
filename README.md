# Minesweeper
The classical game of Minesweeper created with PyGame

## Run the program
`$ python3 Game.py`
* _-c_ : Number of columns.
* _-r_ : Number of rows.
* _-g_ : Disable the GUI.

## File description
* _Game.py_ : Everything revolving the game logic.
* _GameUI.py_ : Everything revolving the UI.
